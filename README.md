## Next steps

 * Clone repository (optional)
 * Run:
 ```
 zig build
 zig build run --release=fast
 # Performance critical, insane request and proccess speeds (only ~700-800ns per request!!)
 ```

 * Or type-safe and memory safe linting and support:
 ```
 zig build run
 ```
 * Open localhost link and run!
